# 1장. 이 책의 목표와 최종 완성본 미리보기

## 1. 목차 개요

- **목적**: 독자가 "무엇을 만들 수 있는지" 먼저 보여주고 동기 부여

## 2. 주요 내용

### 1.1 이 책이 다루는 범위

- **Fine-tuning이 아닌 RAG 파이프라인 튜닝**: 모델 재학습 없이 검색 및 시스템 최적화로 성능 확보
- **비용 효율적 + 데이터 보안 유지**: 로컬 환경 중심의 구축 전략

### 1.2 최종 결과물 데모 시나리오

- **정형+비정형 복합**: "김대리 연차 + 휴가 규정" 통합 답변
- **정형 데이터 조회**: "영업팀 매출 합계" 실시간 쿼리
- **비정형 문서 검색**: "사내 온보딩 절차" 문서 기반 답변
- **근거 포함 요약**: 수치와 텍스트 근거를 포함한 리포트 생성

### 1.3 아키텍처 한 장 요약

- **런타임 에이전트**: LangChain Agent (Router)를 통한 정형/비정형 경로 분기
- **인덱싱 파이프라인**: PDF 파싱(텍스트/LLaVA/OCR)을 통한 멀티모달 벡터화

### 1.4 사용 기술 스택 (상세)

- **추론**: Ollama + DeepSeek R1 / LLaVA / EasyOCR
- **인프라**: FastAPI, PostgreSQL, ChromaDB, LangChain
- **사양**: 최소 RAM 16GB 권장 환경

### 1.5 이 책을 마치면 할 수 있는 것

- 사내 문서 기반 AI 챗봇 구축 실무
- 정형(DB) + 비정형(문서) 데이터 통합 질의응답 시스템 설계
- RAG 파이프라인(PDF 파싱 ~ 튜닝) 전 과정 습득
- 로컬 LLM 기반 보안 강화 모델 운영 능력 확보
