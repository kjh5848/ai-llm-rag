# 4장. 베이스 시스템 설정: "초간단 사내 시스템" 구축

## 1. 목차 개요
- **목적**: 준비된 코드를 `git clone`하여 실습의 기반이 될 기본적인 사내 시스템(DB, UI)을 즉시 확보합니다.

## 2. 주요 내용
### 4.1 베이스 소스 코드 확보 (Git Clone)
- 실습 레포지토리 구성 확인
- 환경변수(.env) 설정 및 의존성 설치
- 시스템 실행: `uvicorn main:app --reload`

### 4.2 데이터 아키텍처 이해 (3테이블)
- **RDB 구조 분석**: AI가 조회하게 될 직원(Employee), 휴가(Leave), 매출(Sales) 데이터 모델의 관계 이해

### 4.3 기본 CRUD UI 확인
- **관리자 기능**: AI 연동 전, 대시보드에서 직접 데이터를 입력하고 조회하는 기본 웹 UI 구조 파악

### 4.4 핵심 API 포인트 점검
- 이후 AI 에이전트가 도구(Tool)로 사용하게 될 기본적인 API 엔드포인트들의 작동 방식 확인
