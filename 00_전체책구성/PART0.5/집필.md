# PART 0.5. 개발 환경 설정

> **"도구의 준비가 프로젝트의 절반이다."**
> AI 프로젝트는 고성능 GPU와 복잡한 라이브러리 설정 때문에 시작부터 난관에 부딪히기 쉽습니다. 이 파트에서는 누구나 자신의 PC(Mac/Windows/Linux)에서 최신 오픈소스 LLM을 구동하고 개발할 수 있도록 **Ollama**와 **Docker**를 활용한 최적의 환경 구축 가이드를 제공합니다.

---

## 0.5.1 하드웨어 및 소프트웨어 필수 요구사항

최신 LLM인 DeepSeek R1과 LLaVA를 원활하게 구동하기 위해 권장되는 사양입니다.

- **CPU**: 최근 3년 내 출시된 멀티코어 프로세서
- **RAM**: **최소 16GB 이상** (32GB 이상 권장)
  - *Tip: 메모리가 부족할 경우 텍스트 LLM과 Vision 모델을 동시에 띄우지 않고 인덱싱과 런타임을 분리하여 운영하는 전략을 사용합니다.*
- **GPU**: NVIDIA GPU (리눅스/윈도우) 또는 Apple Silicon (Mac)
- **저장공간**: 모델 및 데이터베이스 포함 약 **20GB 이상**의 여유 공간
- **OS**: macOS, Windows (w/ WSL2), Ubuntu Linux 22.04+

---

## 0.5.2 로컬 LLM의 허브: Ollama + DeepSeek R1 설치

우리는 복잡한 설정 없이 명령 한 줄로 모델을 관리하기 위해 **Ollama**를 사용합니다.

1. **Ollama 설치**: [ollama.com](https://ollama.com)에서 자신의 OS에 맞는 설치 프로그램을 내려받아 설치합니다.
2. **DeepSeek R1 다운로드**:
   ```bash
   ollama pull deepseek-r1
   ```
3. **작동 확인**:
   ```bash
   ollama run deepseek-r1 "안녕하세요! 당신의 기능을 설명해주세요."
   ```
4. **(선택) GPU 가속 설정**: NVIDIA 사용자의 경우 최신 드라이버가 필수이며, Mac 사용자는 별도 설정 없이 Metal 가속이 이루어집니다.

## 0.5.3 Python 가상환경 및 주요 의존성 구성

프로젝트의 독립성을 유지하기 위해 가상환경을 구축하고 필요한 라이브러리를 설치합니다.

```bash
# 가상환경 생성
python -m venv venv

# 가상환경 활성화 (Unix/Mac)
source venv/bin/activate
# 가상환경 활성화 (Windows)
# venv\Scripts\activate

# 주요 패키지 설치
pip install fastapi uvicorn langchain langchain-ollama chromadb sqlalchemy psycopg2-binary sentence-transformers pypdf python-docx openpyxl
```

## 0.5.4 정형 데이터의 성지: PostgreSQL 설정

회사의 인사 및 매출 정보를 저장할 데이터베이스입니다. 복잡한 로컬 설치 대신 **Docker**를 활용한 실행을 강력히 권장합니다.

```bash
# Docker를 이용한 PostgreSQL 실행 (예시)
docker run --name ai-postgres -e POSTGRES_PASSWORD=mysecretpassword -p 5432:5432 -d postgres:latest
```

설치 후에는 `Database GUI(DBeaver, TablePlus 등)`를 통해 연결이 정상적으로 되는지 확인합니다.

## 0.5.5 프로젝트 초기 구성 및 .env 설정

보안을 위해 API 키와 DB 접속 정보를 프로젝트 코드에서 분리합니다.

1. **레포지토리 클론**: `git clone <your-repo-address>`
2. **환경변수 파일 생성**: `.env.example` 파일을 복사하여 `.env`를 만듭니다.
3. **핵심 설정**:
   ```env
   DATABASE_URL=postgresql://postgres:mysecretpassword@localhost:5432/postgres
   OLLAMA_BASE_URL=http://localhost:11434
   EMBEDDING_MODEL_NAME=ko-sroberta-multitask
   ```

## 0.5.6 주요 의존성 목록 (Snapshot)

우리가 사용하게 될 핵심 도구들의 용도입니다.

- **FastAPI**: 대화형 UI를 제공할 웹 프레임워크
- **LangChain**: LLM과 다양한 툴(DB, 검색)을 연결하는 끈
- **ChromaDB**: 문서의 의미를 벡터로 저장하고 고속 검색
- **EasyOCR / PyPDF**: 문서 파싱 엔진
- **Sentence-Transformers**: 한국어 문장을 숫자로 바꿔줄 임베딩 엔진
