# 🖼️ [실습] 시각 자료(이미지/그래프) 처리 가이드

RAG 시스템을 구축할 때, 텍스트뿐만 아니라 **이미지나 그래프** 안에 담긴 정보를 어떻게 추출하느냐가 매우 중요합니다.

## 1. 실습용 이미지 파일
- **파일명**: `07_2024년_매출추이_그래프.png`
- **내용**: 2024년 1월부터 7월까지의 월별 매출 성장 추이 그래프

## 2. 전처리 실습 과제
이 이미지를 보고 다음 질문에 답변할 수 있도록 전처리(Preprocessing) 엔진을 구성해 보세요.

- **질문 1**: "2024년 7월의 매출액은 얼마인가요?" (이미지 내 텍스트 인식 테스트)
- **질문 2**: "올해 상반기 매출은 어떤 추세를 보이고 있나요?" (그래프 경향 분석 테스트)

## 3. 권장 처리 방식
1. **OCR (Optical Character Recognition)**: Tesseract나 Google Vision API 등을 사용하여 이미지 내 텍스트 직접 추출.
2. **Multimodal LLM (GPT-4o, Llama3-Vision 등)**: 이미지를 직접 입력받아 텍스트로 설명(Captioning) 하거나 표(Markdown Table)로 변환.
3. **캡션 추가**: 이미지 하단에 사람이 직접 설명을 적어두는 방식으로 전처리 품질 향상.
