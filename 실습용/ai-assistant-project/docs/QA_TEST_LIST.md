# 🤖 메타코딩 AI 비서 테스트 질문 리스트

이 문서는 AI 비서의 진화 과정(Rule-based -> Agent)을 검증하기 위한 테스트셋입니다.

---

## 📅 [Part 1~5] 에이전트 도입 전: 하이브리드 검색 (Hybrid Search)

**특징**: 질문을 분석하여 SQL(정형) 또는 Vector(비정형) 검색 중 하나를 선택하거나 단순 결합합니다. (라우터가 규칙 기반 정적 분기)

### 1. 정형 데이터 검증 (SQL Direct)

- **직원 정보**: "김주혁님의 소속 부서와 이메일 주소를 알려줘."
- **조직 현황**: "우리 회사 영업팀에는 총 몇 명의 직원이 근무하고 있나요?"
- **매출 조회**: "최근 영업팀에서 발생한 매출 내역을 요약해줘."

### 2. 비정형 데이터 검증 (RAG w/ VectorDB)

- **연차 규정**: "회사의 연차 휴가 부여 기준에 대해 설명해줘."
- **보안 지침**: "사내 보안 사고 예방을 위한 핵심 지침 3가지를 알려주세요."
- **복리후생**: "우리 회사의 주요 복리후생 제도에는 어떤 것들이 있나요?"

---

## 🚀 [Part 6] 에이전트 도입 후: MCP Agent (Tool Calling)

**특징**: AI가 스스로 필요한 **도구(Tools)**를 판단하고, 순차적 또는 병렬적으로 실행하여 답을 찾아냅니다. (동적 추론)

### 1. 도구 선택 능력 검증 (Tool Selection)

_AI가 적절한 함수를 호출하는지 로그(`AgentService`)를 통해 확인하세요._

| 질문                          | 예상 도구 (Tool)                      |
| :---------------------------- | :------------------------------------ |
| "김대리 휴가 며칠 남았어?"    | `get_leave_balance(name="김대리")`    |
| "이번 달 영업팀 매출 합계는?" | `get_sales_sum(dept="영업팀")`        |
| "보안 규정 문서 찾아줘"       | `search_documents(query="보안 규정")` |

### 2. 복합 추론 검증 (Multi-step Reasoning)

_하나의 질문을 해결하기 위해 여러 도구를 사용해야 하는 상황입니다._

- **휴가 + 규정**:  
  "**김대리** 휴가 며칠 남았고, 우리 회사 **연차 사용 규정**이 어떻게 돼?"  
  👉 `get_leave_balance` 실행 후 결과 확인 -> `search_documents` 실행 -> 정보 종합하여 답변

- **부서 정보 + 매출**:  
  "**영업팀** 직원 명단을 보여주고, 해당 부서의 **최근 매출 실적**도 요약해줘."  
  👉 `list_employees(dept="영업팀")` -> `get_sales_sum(dept="영업팀")` -> 통합 답변
