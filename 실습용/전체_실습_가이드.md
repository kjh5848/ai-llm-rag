# AI 업무 비서(RAG + MCP) 전체 실습 가이드

본 문서는 '사내 문서 기반 AI 업무 비서' 프로젝트의 전체 실습 과정을 안내하는 통합 가이드입니다. 각 단계별 폴더로 이동하여 상세 실습을 진행하시기 바랍니다.

---

## 🏗️ 1. 실습 로드맵 (Hands-on Roadmap)

본 실습은 **[기초 다지기] → [시스템 클론] → [지식 엔진 구축] → [지능형 에이전트 완성]** 의 4단계 흐름으로 구성됩니다.

| 단계                       | 폴더명                   | 학습 내용                              | 핵심 기술           |
| :------------------------- | :----------------------- | :------------------------------------- | :------------------ |
| **Stage 1. 기초 RAG**      | `02_basic_rag`           | LLM의 한계 체감 및 기초 RAG 파이프라인 | LangChain, VectorDB |
| **Stage 2. 인프라 구축**   | `04_base_system`         | 사내 DB(직원/휴가) 환경 구축 및 연동   | SQLite/PostgreSQL   |
| **Stage 3. 지식 엔진**     | `06_vectordb` ~ `07_rag` | PDF 문서 디지털화 및 전문 Q&A 엔진     | ChromaDB, Embedding |
| **Stage 4. 통합 에이전트** | `08_agent` ~ `10_tuning` | MCP(DB)와 RAG(문서) 통합 및 UI 연결    | Agent Routing, UI   |

## 💻 2. 사전 준비 (Environment Setup)

실습을 시작하기 전, 아래 환경이 반드시 갖춰져 있어야 합니다. (상세 내용은 `03_setup` 폴더 참조)

1.  **AI 모델**: [Ollama](https://ollama.com) 설치 후 아래 모델 다운로드
    - `ollama pull deepseek-r1:8b` (추론형 생성 모델)
    - `ollama pull nomic-embed-text` (임베딩 모델)
2.  **프로그래밍 환경**: Python 3.10 이상 권장
3.  **필수 라이브러리 설치**:
    ```bash
    pip install langchain langchain-community langchain-ollama langchain-chroma
    ```

## 📂 3. 단계별 실습 안내

### Phase 1: 기초 RAG 실습 (`02_basic_rag`)

- **목표**: "AI는 왜 우리 회사 정보를 모를까?"라는 의문에서 시작하여, 외부 지식을 주입하는 **RAG(Retrieval-Augmented Generation)** 의 핵심 원리를 배웁니다.
- **주요 파일**: `step1_fail.py` (실패 사례) ~ `step4_rag.py` (성공 사례)

### Phase 2: 베이스 시스템 연동 (`04_base_system`)

- **목표**: 실제 사내에 구축된 인사 정보 DB나 근태 관리 시스템의 구조를 이해하고, AI가 직접 DB에 접근할 수 있는 **MCP(Model Context Protocol)** 기반 환경을 준비합니다.
- **주요 파일**: `db_setup.py`, `mcp_server.py`

### Phase 3: 지식 검색 엔진 구축 (`06_vectordb`, `07_rag_engine`)

- **목표**: 사내 규정집(PDF)이나 업무 매뉴얼을 AI가 검색할 수 있는 형태로 변환합니다. 텍스트 추출, 청킹, 임베딩의 전 과정을 코드로 구현합니다.
- **핵심 기술**: 문서 로더(`PyPDFLoader`), 벡터 DB(`Chroma`)

### Phase 4: 지능형 에이전트 및 UI (`08_agent`, `09_ui`)

- **목표**: 사용자의 질문이 "규정"에 관한 것인지 "DB 데이터"에 관한 것인지 AI가 스스로 판단하여 적절한 도구를 사용하는 **라우팅(Routing)** 에이전트를 완성합니다.
- **최종 결과물**: 웹 브라우저에서 동작하는 실전 AI 업무 비서 UI

## 🚀 실습 시작하기

1.  본 저장소를 클론한 후 `실습용` 폴더로 이동합니다.
2.  가장 먼저 `02_basic_rag` 폴더의 가이드를 따라 RAG의 맛을 느껴보십시오.
3.  이후 각 장의 흐름에 맞춰 폴더별 가이드를 순차적으로 따라가면 됩니다.

> **주의**: 실습 중 오류가 발생할 경우 `03_setup` 폴더의 환경 점검 가이드를 다시 한번 확인해 주시기 바랍니다.
