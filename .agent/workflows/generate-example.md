# 💻 예제 코드 생성 및 검증 워크플로우 (디렉토리 한정)

이 워크플로우는 특정 실습 디렉토리(예: `실습용/ex01`) 내에 동작 가능한 코드를 만들고 터미널에서 검증할 때 사용합니다.

## 사전 준비
작업할 실습 디렉토리 경로를 확인하세요. (예: `cd 실습용/ex01`)

## 진행 순서

1. **코드 스펙 분석**: 챕터 본문(예: `집필/01/chapters/CH01...md`)의 내용을 바탕으로 어떤 예제 코드가 필요한지 파악합니다.
2. **모듈화된 코드 생성**: 
   - `example-code-generator` 스킬을 사용하여 단일 파일이 아닌 `src/`, `config/` 등 프로덕션 레벨의 폴더 구조를 만듭니다.
   - `README.md` 와 `requirements.txt` 도 함께 생성하며, 주석은 한국어로 상세히 추가합니다.
3. **가상환경 준비 및 패키지 설치**: 
   - 터미널을 열고 해당 실습 디렉토리로 이동한 후, Python 가상환경(venv 등)을 활성화합니다.
   - `requirements.txt` 에 명시된 패키지를 설치합니다.
4. **실행 및 예외 검증**: 
   - `code-verifier` 스킬을 사용하여 코드를 직접 실행합니다. 
   - 파일 권한, 입력 데이터 누락 등 다양한 에지 케이스(Edge Case)를 시뮬레이션하여 에러가 제대로 잡히는지 디버깅합니다.
5. **결과 리포팅**: 검증에서 통과된 로그와 최종 수정된 코드를 바탕으로 책 원고에 들어갈 '코드 워크플로우' 설명 자료를 정리합니다.
