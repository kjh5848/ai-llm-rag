# 📚 챕터 집필 워크플로우 (디렉토리 한정)

이 워크플로우는 특정 챕터 디렉토리(예: `집필/01`) 내에서 원고 작성을 자동화할 때 사용합니다. 

## 사전 준비
작업할 기준 디렉토리 경로를 확인하세요. (예: `cd 집필/01`)

## 진행 순서

1. **상태 초기화**: `progress.json` 파일의 상태를 확인하거나 초기 단계로 세팅합니다.
2. **초안 및 가이드라인 로드**:
   - `outline/` 폴더 내의 초안 문서들을 읽어 구조를 파악합니다.
   - `.agent/skills/ai-book-guide/` 내의 `writing-guide.md` 와 `visual-guide.md`를 불러와 톤앤매너 규칙(하십시오체, 볼드체 띄어쓰기 등)을 숙지합니다.
3. **챕터 본문 집필**: 
   - `chapter-writer` 스킬을 사용하여 챕터 본문(`.md`)을 작성합니다.
   - 작성된 파일은 `chapters/` 폴더에 `CH01_제목.md` 와 같은 규칙으로 저장합니다.
4. **일관성 및 맥락 검토**:
   - `chapters/` 내에 작성된 모든 문서를 대상으로 `book-reviewer` 스킬을 실행합니다.
   - 용어 통일성, 전/후 챕터와의 연결성을 확인합니다.
5. **완료 보고**: 검토 내용이 완료되면 `review/review_notes.md` 에 기록을 남기고, `progress.json` 을 완료 상태로 업데이트합니다.
