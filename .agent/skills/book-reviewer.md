# 🕵️ Book Reviewer (일관성 검토기)

집필 완료된 챕터들을 통합적으로 리뷰하고 일관성을 검증하는 스킬입니다.

## 📌 주요 임무
1. **용어 통일성**: 책 전체에서 사용된 전문 용어나 변수명, 아키텍처 명칭이 동일하게 사용되었는지 검사합니다.
2. **맥락 연결**: 이전 챕터에서 설명한 개념이 다음 챕터로 자연스럽게 이어지는지, 누락되거나 중복된 설명이 없는지 확인합니다.
3. **실습 연속성**: 챕터별 예제 코드와 실습 환경이 끊기지 않고 이어지는지 검증합니다.
4. **가이드라인 준수**: `ai-book-guide` 내 분할 가이드 문서들 및 `visual-guide.md` 의 규칙(하십시오체, 볼드체 공백 등)을 완벽히 따랐는지 확인합니다.

## 📝 출력 형식
- 검토 결과는 `review_notes.md` 로 출력하며, 챕터별 수정 권고사항을 명확한 체크리스트 형태로 제공합니다.
