# 🔧 Code Verifier (코드 검증기)

`example-code-generator` 가 만든 예제 코드가 독자 환경에서 오류 없이 실행되는지 테스트하는 스킬입니다.

## 📌 주요 임무
1. **독자 환경 시뮬레이션**: Python 프로젝트의 경우 터미널을 열어 **가상환경(venv)에 접근** 한 후 코드를 실행합니다.
2. **의존성 검증**: `requirements.txt` 에 명시된 패키지들만 설치하여 코드가 동작하는지 확인합니다. 모자란 라이브러리가 있다면 추가합니다.
3. **에지 케이스 테스트**: 입력값이 없거나 파일 경로가 잘못된 경우 등 예외 상황에 대한 에러 처리가 잘 되어 있는지(독자에게 친절한 에러 메시지가 나오는지) 확인합니다.
4. **결과물 리포팅**: 검증이 완료되면 집필 에이전트가 참고할 수 있도록 실행 결과 로그와 최종 수정된 코드 내역을 리포트합니다.

## ⚠️ 필수 준수 규칙
- 검증 시 발생하는 모든 오류 원인과 해결 방법을 주석 또는 리포트 문서에 한국어로 기록합니다.
- 반드시 가상환경(venv 등)을 활성화한 상태에서 패키지 설치 및 실행 테스트를 진행합니다.
