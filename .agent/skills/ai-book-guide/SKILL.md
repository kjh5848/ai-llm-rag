# 📚 기술 도서 집필 및 시각화 통합 가이드 V3.0

이미지 생성 프롬프트, 아이콘 스타일, 파일 삽입 규칙은 아래 문서를 따릅니다.

- **바로가기**: [visual-guide.md](./visual-guide.md)
- **주요 내용**: `16:9` 비율, 미니멀 라인 아트, 실제 이미지 경로 삽입 규칙.

## 2. ✍️ 원고 집필 시스템 (Writing System)

문체, 코드 작성 표준, 실습 구성 방법은 아래 문서를 따릅니다.

- **바로가기**: [writing-guide.md](./writing-guide.md)
- **주요 내용**: 집필 핵심 전략, 설명의 기술, 하십시오체, 챕터 마무리 규칙, 코드 설명 표준.
