# 💻 Example Code Generator (예제코드 생성기)

책의 독자가 직접 실행해볼 수 있는 실습용 코드를 작성하는 스킬입니다.

## 📌 주요 임무
1. **모듈화된 구조 설계**: 단일 파일에 모든 코드를 넣는 것을 지양하고, 실제 프로덕션 도입이 가능할 정도의 모듈 구조(예: `src/`, `config/`, `utils/`)를 잡습니다.
2. **가독성 및 주석**: 모든 코드의 주석은 **반드시 한국어** 로 작성하며, 독자가 이해하기 쉽도록 상세히 적습니다.
3. **PEP8 및 컨벤션**: Python 코드의 경우 PEP8을 준수하며, 명확한 타입 힌트(Type Hint)를 포함합니다.
4. **실행 가능한 단위**: 각 챕터/실습 번호(예: `ex01`) 로 디렉토리를 분리하고, 해당 디렉토리 내에서 즉시 실행 가능하도록 `README.md` 와 `requirements.txt` 를 함께 생성합니다.

## ⚠️ 필수 준수 규칙
- 복잡한 로직 앞에는 반드시 해당 로직의 목적을 설명하는 블록 주석을 작성합니다.
- 터미널 실행 명령어 가이드를 제공할 때, Python 가상환경(venv 등) 활성화 과정을 포함합니다.
